The issue is that the main game needs a local server.

🚀 How to Run GestureX Racing:

  1. Close this page
  2. Double-click START_GAME.bat
  3. OR open Command Prompt in this folder
  4. Run: python -m http.server 8000
  5. Open browser to: http://localhost:8000

⚠️ Important: The game MUST run on a server (localhost) because it loads 3D models (.glb files) which require HTTP protocol.
